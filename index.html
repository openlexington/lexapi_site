LEXAPI

LEXAPI is a JSON API similar to and heavily inspired by PDXAPI and others that provides geographic and real-time data from Lexington, KY. Currently, most of the data is user contributed as the Lexington Fayette County Urban Government currently does not publish real-time open data. The LEXAPI is a an open source project developed by OpenLexington.

Usage

The API is still in development.
